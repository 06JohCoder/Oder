## 🚀 Hướng dẫn cài đặt & chạy dự án

 1. Clone project
```bash
git clone obj
cd obj
```
2. Cài đặt & chạy Backend

- cd backend
- npm install
- npm start
  
3. Cài đặt & chạy Frontend
- cd frontend
- npm install
- npm start

4. Kết nối cơ sở dữ liệu NoSQL (MongoDB)
   MONGO_URI=mongodb://localhost:27017/<ten_database>  (ở đây tên data của tôi là productsUser)
